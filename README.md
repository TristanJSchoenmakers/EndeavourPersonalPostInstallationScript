My personal dotfiles


## Getting started


#### 1 - Install Arch Linux


#### 2 - Clone repository
``` bash
git clone https://github.com/TristanJSchoenmakers/dotfiles.git
```


#### 3 - Move directory
``` bash
mv dotfiles .config
```


#### 4 - update permissions install script
``` bash
sudo chmod +x ~/.config/install.sh
```


#### 3 - Run the script
``` bash
sudo bash ~/.confiq/install.sh   
```
